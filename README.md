# JobTracker

A personal web application to track job applications.

## Tech Stack

- **ASP.NET Core MVC** – Web framework
- **.NET 9**
- **Entity Framework Core** – ORM
- **SQLite** – local database
- **ASP.NET Core Identity** – authentication
- **Bootstrap 5** – UI styling
- **Razor Views**

## Features

- Create, edit, view and delete job applications
- Track application status (Applied, Interview, Offer, Rejected)
- Dashboard with application statistics
- Search and filter applications by company, job title and status
- CSV export of all or filtered job applications
- Authentication (login, register, logout)
- Responsive UI
